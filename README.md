# Рок-фестиваль «Звуки Свободы»

Этот проект представляет собой официальный лендинг для рок-фестиваля «Звуки Свободы». Сайт создан для информирования посетителей, продажи билетов и сбора отзывов. Проект разработан как учебный и демонстрирует навыки верстки, создания интерактивных элементов и работы со сборщиками.

## Использованные технологии

Проект построен на **чистом HTML, CSS и JavaScript** без использования тяжелых фреймворков, что делает его легким и быстрым.

- **HTML5** — семантическая разметка.
- **CSS3** — кастомные стили, анимации, медиа-запросы для адаптивности.
- **Bootstrap 5** — используется для сетки, компонентов навигации (Navbar) и офф-канвас корзины.
- **JavaScript (ES6)** — обеспечивает всю интерактивность: слайдер, корзину, модальное окно и управление отзывами.
- **Google Fonts** — подключение шрифтов `Montserrat` и локального шрифта `ST-SimpleSquare`.

## Функциональность и интерактивность

### 1. Автоматический слайдер
- На главном экране реализован слайдер с тремя изображениями.
- Слайды переключаются автоматически каждые **10 секунд**.
- Управление также осуществляется через точки-индикаторы внизу слайдера.

### 2. Корзина для билетов
- **Три тарифа**: Обычный, Фан и VIP билеты.
- При нажатии на кнопку «Добавить в корзину» открывается офф-канвас панель.
- В корзине реализован функционал:
  - Увеличение/уменьшение количества билетов.
  - Автоматический пересчет итоговой суммы.
  - Удаление позиций при уменьшении до 0.
  - Кнопка «Перейти к оплате» с очисткой корзины и выводом подтверждения.

### 3. Управление отзывами
- Отображение списка отзывов с фотографией пользователя.
- **Удаление** любого отзыва по нажатию на крестик (с подтверждением).
- **Добавление** нового отзыва через модальное окно:
  - Поля: Имя, Текст отзыва, Загрузка фото.
  - Обязательное согласие на обработку данных.
  - Кнопка отправки активна только при заполнении всех обязательных полей.
  - Для фото реализована валидация размера (не более 2 Мб) и формата.

## Адаптивность

Сайт корректно отображается на всех типах устройств:

- **Десктопы** (стандартное разрешение) — полноценный вид с горизонтальным расположением карточек и слайдером.
- **Планшеты** (768px) — оптимизирована навигация, блоки артистов и отзывов становятся более компактными.
- **Мобильные устройства** (480px и менее) — все блоки выстраиваются в колонку, меню сворачивается в "гамбургер", контент адаптируется под маленький экран.
